===== Observable<T>
* 요소가 가지고 있는 이벤트를 계속 방출함
* Subcribers가 방출한 이벤트들에 대해 반응할 수 있음

===== Observable의 Life Cycle 
* Next 
** Interger 또는 Tap
* Error 
** Observable 종료됨
* Completed 
** Observable 종료됨

image:./image/observable-life-cycle.png[45%, 45%]

===== Observable Events
* 열거형으로 구성됨

[source, swift]
----
public enum Event<Element> {
    case next(Element)
    case error(Swift.Error)
    case completed
}
----
