===== Tip
* 단축키 `Command + 0` 을 누르면 캔버스 영역이 100% 맞춰짐
* 단축키 `Command + 1` 을 누르면 Canvas 영역의 아트보드들이 View 영역으로 한 눈에 볼 수 있음
* 단축키 `Command + 2` 을 누르면 선택한 아트보드나 작업물 영역으로 포커스가 이동함
* 단축키 `Command + +` 을 누르면 Zoom In
* 단축키 `Command + -` 을 누르면 Zoom Out
